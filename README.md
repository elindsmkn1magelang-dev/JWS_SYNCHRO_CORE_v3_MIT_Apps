# JWS Synchro Core - Master Side (v1.2)
Proyek Jam Waktu Sholat otomatis berbasis ESP8266 dengan sinkronisasi NTP dan penyimpanan EEPROM.

### Fitur Utama:
* **GPS Sync & EEPROM**: Menyimpan koordinat lokasi secara permanen.
* **NTP Time**: Update waktu otomatis via internet.
* **NeoPixel Display**: Animasi rainbow dan transisi halus.
* **DFPlayer Mini**: Otomasi suara Tarhim sebelum waktu sholat.

### Komponen:
1. Wemos D1 Mini (ESP8266)
2. RTC DS3231
3. P10 LED Panel / NeoPixel Strip
4. DFPlayer Mini

### Cara Penggunaan:
1. Upload kode ke Wemos D1 Mini.
2. Hubungkan ke Hotspot "JWS-SMK-ENGINEERING".
3. Gunakan APK MIT App Inventor untuk set lokasi.
